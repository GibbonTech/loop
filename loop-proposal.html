Loop

Vous faites du portage salarial pour chauffeurs VTC.

Vous gérez déjà tout : contrats, paie, cotisations, fiscalité. Le métier est là. Ce qui manque, c'est l'interface.

Un endroit où vos chauffeurs voient leurs revenus, téléchargent leurs documents, soumettent leurs frais. Sans vous appeler. Sans WhatsApp.

Le besoin

Un portail chauffeur — web et mobile.

  • Dashboard — revenus par jour, semaine, mois, par plateforme
  • Répartition — CA → indemnités km → brut → net
  • Sync plateformes — Uber, Bolt, Heetch centralisés
  • Bulletins de paie — PDF téléchargeables
  • Notes de frais — upload factures, récup TVA
  • Documents — contrat, mutuelle, attestations
  • Profil — infos chauffeur + véhicule

Plus un back-office pour vous. Upload des fiches de paie, validation des frais, gestion des chauffeurs.

Comment

Le chauffeur connecte ses comptes VTC via un widget sécurisé. Ses revenus remontent automatiquement. Pas de saisie manuelle.

Une seule base de code. Deux interfaces : web pour l'ordi, app native pour iOS et Android.

Vous uploadez les bulletins de paie quand ils sont prêts. Le chauffeur reçoit une notif, télécharge en un clic.
